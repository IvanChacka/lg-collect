# 自动化热点追踪与深度内容生成工作流 (修订版)

## ⚙️ 全局配置 (Global Settings)
* **核心驱动模型**：`minimax`
* **API Key**：通过环境变量配置（不要把密钥写进仓库）
* **通信主干**：飞书 OpenAPI（发消息）+ 事件订阅 WebSocket（收用户 @选择）

---

## 第一阶段：热点发现与素材收集 (信息源构建)
此阶段旨在全面获取互联网最新热点，并**必须提取实质性内容**，拒绝只抓标题的表面功夫。

* **热点收集**：
    * 调用第三方数据服务（如今日热榜 API、聚合数据等）获取“微博热搜”、“抖音热点榜”、“B站热门”。
    * **初步过滤**：让 minimax 模型快速判定，保留具备“科技”或“医疗健康”延展潜力的社会热点，剔除纯饭圈八卦。
* **素材深度收集（文本+视频双管齐下）**：
    * **文本类**：通过搜索引擎插件抓取该热点事件的前 5-10 篇高赞/权威新闻或文章。
    * **视频类（硬性要求）**：必须获取视频的完整口播/字幕文案。
        * *技术方案建议*：利用开源解析工具（如 `BBDown`）拉取视频字幕或音频，音频再做 ASR 转写。
        * 若原视频无字幕：调用 OpenAI Whisper 或同类开源语音转文字模型，将提取出的视频音频强制转录为文本，作为后续 AI 的分析基底。

## 第二阶段：科技与医疗方向的深度挖掘 (AI 思考核心)
不要停留在热点表象，核心逻辑是：**“老百姓看到了这个热点，脑袋里会有什么疑问？我们用科技/医疗视角去解答。”**

* **思考模型示例（风筝节效应）**：
    * *大众视角*：潍坊风筝节火了，风筝好大好酷。
    * *大模型挖掘方向*：“巨型风筝的空气动力学原理（科技）”、“被风筝线割伤后的紧急医疗缝合与组织再生（医疗）”、“无人机如何规避风筝网络（科技/产业）”。
* **Minimax Prompt (提示词) 设计思路**：
    > “你是一个极具洞察力的科技与医疗行业科普作家。我将提供一组关于近期热点【XXX】的原始素材。请站在普通大众的视角，思考他们看到该事件后会产生哪些好奇或担忧。
    > 请忽略情绪化表达，以此为切入点，深度挖掘背后的科技原理（如材料学、物理学、AI应用）或医疗健康知识（如病理机制、公共卫生干预、急救措施）。输出 3 个具有极强反差感或知识增量的选题切入点。”

## 第三阶段：生成初版大纲与选题 (AI 内容生成)
要求 Minimax 将第二阶段的思考，格式化输出为供人工决策的“菜单”。

* **输出格式要求**（生成 3 个选项，每个包含）：
    1.  **吸睛标题**：制造好奇心，如“潍坊天上飞的不是风筝，是空气动力学的‘流氓’”
    2.  **核心论点**：一句话概括你想解答的大众疑问。
    3.  **大纲框架**：引子（热点引入） -> 提出核心疑问 -> 深度科普拆解 -> 行业/生活启示。

## 第四阶段：飞书交互与人工选择（WebSocket + Human-in-the-loop）
利用现成的基础设施完成人机交互。

* **执行动作**：
    * 调用飞书 OpenAPI 发送一条可渲染 Markdown 的消息（包含当前热点与选项列表）。
    * 通过飞书“事件订阅 WebSocket”监听群内用户 **@机器人** 的回复。
* **格式与交互设计**：
    * 选项按序号（1/2/3）+ 字母（A/B/C）展示。
    * 程序挂起（如使用 LangGraph 的 interrupt），等待用户 @回复的序号/字母，再 resume 继续跑后续节点。

## 第五阶段：生成深度长文 (终版输出)
接收到选择指令后，将选中选题的大纲、第一阶段收集的全部真实素材喂给 Minimax，生成最终交付物。

* **文章调性要求**：
    * **非脚本格式**：摒弃分镜头/双列脚本格式，输出为一篇完整的高质量深度阅读文章。
    * **故事化开局**：最前面必须极其吸引人，像讲故事、悬疑片或引人入胜的纪录片开场，迅速抓住读者眼球。
    * **深度科普风格**：正文部分语气要像《博物》或《科技袁人》，通俗易懂但专业名词绝对严谨，层层递进剥开现象看本质。
* **Minimax Prompt (提示词) 设计思路**：
    > “用户已选定选题【A】。请根据我之前提供的所有热点事实素材，撰写一篇 3000 字左右的深度科普文章。
    > **核心要求**：
    > 1. 开头必须像讲故事一样，极具画面感和悬念，迅速引人入胜。
    > 2. 结合大众的好奇心，用通俗幽默的语言解释硬核的【科技/医疗】原理。
    > 3. 专业术语必须准确，逻辑层层递进。
    > 4. 结尾升华主题，给出具有现实指导意义的行动建议或未来展望。”
